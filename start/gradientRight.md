# 색상 흐름 연출 만들기
색상이 흐르는 연출입니다.
여러 가지 색상을 추가할 수 있고 흐르는 방향, 속도를 수정해 다양하게 연출할 수 있습니다. 

원 아이콘 대각선 흐름을 만들었던 시나리오에 색상 흐름 연출을 추가하는 예제입니다.

<img src="../../img/2.jpg" width="1000"/>

## 시나리오 편집
홈 화면의 목록에서 생성된 시나리오를 편집할 수 있습니다.

기존 시나리오에 SCENE을 추가하기 위해 해당 시나리오의 `편집` 버튼을 클릭합니다.

<img src="../../img/2-1.jpg" width="1000" style="border: 1px solid"/>

## SCENE 추가 
`새로운 SCENE 추가` 버튼을 눌러 새로운 SCENE을 추가합니다.

<img src="../../img/2-2.jpg" width="1000" style="border: 1px solid"/>

## SCENE 설정
SCENE 설정 화면은 아래와 같습니다.
순서에 맞게 SCENE 연출 설정을 하고 `미리보기`로 설정 내역을 확인합니다.
설정이 완료되면, `저장` 버튼을 눌러 시나리오에 새로운 SCENE 을 추가합니다.

<img src="../../img/2-3.jpg" width="1000" style="border: 1px solid"/>

### 1. 연출 타입 선택
MFPlayer의 기본 연출 타입을 선택합니다.
**파일 재생**, **이미지 흐름**, **색상 흐름**의 연출 타입이 있습니다.
색상을 설정하기 위해서 `색상 흐름`을 선택합니다.

### 2. 흐름 타입 선택
색상의 흐름 타입을 선택합니다.
**오른쪽**, **왼쪽**, **위**, **아래**의 연출 타입이 있습니다.
색상의 오른쪽 흐름을 위해서 `오른쪽`을 선택합니다.

### 3. 아이콘 선택 
색상을 **추가**, **변경**, **삭제** 할 수 있고 두 번째 색상 앞에 추가하기 위해 두 번째 색상을 선택한 뒤 `추가`버튼을 눌러 색상을 추가합니다.

<img src="../../img/2-4.jpg" width="1000" style="border: 1px solid"/>

### 4. SCENE 미리보기
SCENE 편집 완료 후 `미리보기` 버튼을 누르면 재생 화면이 나타나 편집 결과를 확인할 수 있습니다.
`미리보기 정지`로 재생 화면이 닫힙니다.

원하는 형태의 연출이 될 때까지 `미리보기`를 통해 반복적으로 빠른 수정 - 확인이 가능합니다.

<img src="../../img/2-5.jpg" width="1000" style="border: 1px solid"/>


## SCENE 저장
`저장` 버튼을 누르면 설정된 SCENE 저장되고 시나리오에 추가됩니다.

<img src="../../img/2-6.jpg" width="1000" style="border: 1px solid"/>

시나리오에 SCENE이 추가된 것을 확인할 수 있습니다.

<img src="../../img/2-7.jpg" width="1000" style="border: 1px solid"/>

## 시나리오 저장
`저장` 버튼을 눌러 시나리오를 저장합니다.

<img src="../../img/2-8.jpg" width="1000" style="border: 1px solid"/>

시나리오 목록에서 SCENE을 누르면 선택한 시나리오 내에 포함된 SCENE들이 표시되어 추가된 SCENE을 확인할 수 있습니다.
